Exporting the Config

Export your configuration for use in other files:

Example 
// config.js (or any other file)
import Base from "sin/main/base.js"

const sin = new Base("MyModule", "colors.json", "My Module Config")
    .addButton({
        category: "General",
        configName: "exampleButton",
        subcategory: "General",
        title: "Example Button",
        description: "This is an example button",
        onClick: () => print("Button clicked!")
    })
    .addSwitch({
        category: "General",
        configName: "exampleSwitch",
        subcategory: "General",
        title: "Example Switch",
        description: "This is an example switch"
    })

export default () => sin.settings

Importing the Config in Other Files

Import your configuration in other parts of your application:

Example 
import Config from "./config.js";

// Use Config() if you did export default () => sin.settings
// Use Config if you did export default sin.settings
// I'll continue using Config() for the rest of the example

register("command", () => {
    Config().getConfig().openGui() // Opens the Config GUI
}).setName("openGui")

register("worldLoad", () => {
    ChatLib.chat(Config().exampleValue) // Sends the exampleValue's value in chat
})

Config().getConfig()
    .onOpenGui(() => ChatLib.chat("GUI opened.")) // Sends a message on GUI open
    .onCloseGui(() => ChatLib.chat("GUI closed.")) // Sends a message on GUI close
    .registerListener("exampleValue", (oldV, newV) => ChatLib.chat(`${oldV} -> ${newV}`)) // Message when exampleValue changes

Additional Methods

  • GUI Methods
    .getGui() Returns GUI instance
    .getHandler() Returns handler instance
    .openGui() Opens GUI
    .closeGui() Closes GUI
    .setSize(width, height) Sets GUI size
    .setRatio(ratio) Sets GUI ratio
  • Event Methods
    .onOpenGui(func) Event Handler
    .onCloseGui(func) Event Handler
  • Configuration Methods
    .setValue(key, value) Sets SinGUI value
    .setConfigValue(key, value) Sets config value

Feature Manager

Example 
import Config from "../config"
import { FeatureManager } from "../../tska/event/FeatureManager"

const FeatManager = new FeatureManager(Config().getConfig())
const example = FeatManager.createFeature("exampleConfigVal")

example
  .onRegister(() => ChatLib.chat("registered"))
  .register("stepFps", () => ChatLib.chat("meow"), 1)
  .onUnregister(() => ChatLib.chat("unregistered"))
